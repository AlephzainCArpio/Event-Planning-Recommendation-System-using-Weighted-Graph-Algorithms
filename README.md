# Event Planning Recommendation System using Weighted Graph Algorithms

## 📌 Overview
This project is a web-based event planning system that provides personalized recommendations for venue selection, catering, and logistics. It uses weighted graph algorithms to optimize decision-making based on user preferences.

## 🎯 Objectives
- To simplify the event planning process  
- To reduce manual effort in selecting event services  
- To provide optimized recommendations based on user input such as budget, location, and number of attendees  

## ⚙️ Features
- User input for event preferences (budget, attendees, location)  
- Recommendation system for:
  - Venue selection  
  - Catering services  
  - Logistics planning  
- Basic responsive web interface  
- Data storage using MySQL  

## 🧠 Algorithm Used
This system applies a **weighted graph algorithm**, where:
- Nodes represent event options (venues, catering, logistics)  
- Weights represent factors such as cost, distance, and suitability  
- The system calculates the most optimal combination based on user input
